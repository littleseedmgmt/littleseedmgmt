# Design System

## Design Philosophy

LittleSeedMgmt follows a **clean, minimalist design** inspired by Robinhood and modern state planning applications. The focus is on:

1. **Clarity**: Information should be immediately understandable
2. **Simplicity**: Remove unnecessary visual clutter
3. **Efficiency**: Users should accomplish tasks with minimal clicks
4. **Consistency**: Same patterns and components throughout the system
5. **Accessibility**: WCAG 2.1 AA compliance

## Animation Guidelines

```
DURATION
- Micro interactions: 150ms
- Standard transitions: 200ms
- Page transitions: 300ms
- Complex animations: 400ms

EASING
- Default: ease-out (for entering elements)
- Exit: ease-in (for leaving elements)
- Movement: ease-in-out (for position changes)

PRINCIPLES
- Keep animations subtle and purposeful
- Avoid animations that block user interaction
- Reduce motion for users who prefer it
```

## Accessibility Requirements

1. **Color Contrast**: Minimum 4.5:1 for normal text, 3:1 for large text
2. **Focus States**: Visible focus rings on all interactive elements
3. **Keyboard Navigation**: All features accessible via keyboard
4. **Screen Readers**: Proper ARIA labels and semantic HTML
5. **Reduced Motion**: Respect `prefers-reduced-motion` preference
6. **Touch Targets**: Minimum 44x44px for mobile touch targets
